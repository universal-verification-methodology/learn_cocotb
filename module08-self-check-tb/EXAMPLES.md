## Prompts

1. Restate the core idea of **Self-check pattern** in one sentence.
2. In your own words, capture this beat: *Before scoreboards and agents, every testbench needs a self-check loop: apply stimulus, compute expected from a reference model, sample the DUT output, then pass or fail loudly. cocotb coroutines do the same job with…*
3. Write stimulus → reference expect → compare for and2 with a=1,b=1,y=1.
4. Name what goes wrong if you only `$display` without comparing.
