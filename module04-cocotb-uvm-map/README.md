3. Optional self-check: `./scripts/module.sh 04 --check` (from course root).

1. Local: [http://127.0.0.1:8080/tools/cocotb-uvm-map/index.html](http://127.0.0.1:8080/tools/cocotb-uvm-map/index.html)
