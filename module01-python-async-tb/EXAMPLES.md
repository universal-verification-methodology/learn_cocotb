# Examples — Python async TB

Track A prompts. Browser lab **`python-async-tb`** is shipped.

## Prompts

1. Restate the core idea of **Python async TB** in one sentence.
2. In your own words, capture this beat: *cocotb tests are Python coroutines. You mark a test with async def, register it with the cocotb test decorator, and use await to suspend until the simulator resumes you—often after a timer or a clock edge. That shape…*
3. Name the three pieces: async def, await, and the cocotb test decorator.
4. Sketch a tiny async test that awaits a timer, then pokes one DUT signal (on paper is fine).
