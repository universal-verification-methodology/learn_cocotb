# Examples — cocotb ↔ UVM roles

Track A prompts. Browser lab **`cocotb-uvm-map`** is shipped.

## Prompts

1. Restate the core idea of **cocotb ↔ UVM roles** in one sentence.
2. In your own words, capture this beat: *Plain cocotb is Python driving a simulator with async triggers and DUT handles. pyuvm adds UVM-shaped components on top. SystemVerilog UVM is the same job in another language—tests, drivers, agents, and scoreboards.…*
3. Map dut handle ↔ vif, await ↔ edge, Python check ↔ scoreboard.
4. Say what cocotb does not replace in a full UVM env.
